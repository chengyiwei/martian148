# 简单二维图表

图表对象是 Figure，而 Axes 对象包含在 Figure 里面，表示某一个图表的绘制区域

使用 add_axes 方法可以在 Figure 中添加新的 Axes，然后使用元组对 Axes 对位置和大小进行指定

import matplotlib.pyplot as plt
import numpy as np

x = np.linspace(0, 2*np.pi, 100)
y = np.sin(x)

fig = plt.figure()

# 坐标(0.15, 0.1)，宽70%，高80%
ax = fig.add_axes((0.15, 0.1, 0.7, 0.8))

ax.plot(x, y)
