# 感知机

感知机接受多个输入信号，输出一个信号

x1,x2x_1,x_2x1​,x2​ 是输入信号，yyy 是输出信号，w1,w2w_1,w_2w1​,w2​ 是权重，当 w1x1+w2x2w_1x_1+w_2x_2w1​x1​+w2​x2​ 到达阈值 θ\thetaθ 的时候，输出 111 ，也称为神经元被激活

我们用数学式子表示是 $$ \begin{align} y & = \left{\begin{array}{ll} 0 & \left(w_{1} x_{1}+w_{2} x_{2} \leqslant \theta\right) \ 1 & \left(w_{1} x_{1}+w_{2} x_{2}>\theta\right) \end{array}\right. \end{align} $$

我们可以尝试用感知机来表示与门电路，当 (w1,w2,θ)=(0.5,0.5,0.7)(w_1,w_2,\theta)=(0.5,0.5,0.7)(w1​,w2​,θ)=(0.5,0.5,0.7) 的时候，能和与门的输入输出保持一致

def AND(x1, x2):

# 多层感知机

思考一下异或门，会发现，感知机表示不了异或门

但是通过叠加层，就可以表示出异或门

def XOR(x1, x2):
    s1 = NAND(x1, x2)
    s2 = OR(x1, x2)
    y = AND(s1, s2)
    return y
